# ocr_infer

#### 介绍
* 简介：OCR 离线推理引擎
* 功能：处理大规模复杂网络图像，并从中提取关键词，系统包括文字检测、文字识别、关键词匹配三大功能
* 技术：
  * 使用 TensorRT 量化、加速；
  * 使用流水线技术将系统内部模块并行化，以提高系统的吞吐量；
  * 文字检测使用 DB 模型、文字识别使用 CRNN 模型；
  * 关键词匹配使用 AC 自动机多模匹配。

#### 软件架构
系统分成文字检测、检测后处理、缓存、文字识别、识别后处理、关键词匹配六个模块。
